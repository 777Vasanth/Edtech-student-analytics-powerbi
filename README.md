🚀 Interactive Power BI dashboard analyzing student behavior, course performance, and revenue trends in an EdTech platform

## 📌 Overview

This project presents a comprehensive Student Analytics Dashboard built using Power BI to analyze enrollment patterns, learner behavior, and course performance.

The dashboard enables data-driven decision-making by identifying trends in student engagement, completion rates, and revenue generation.

---

## 🎯 Key Metrics

* Total Enrollments: 100K
* Total Revenue: ₹199M+
* Average Rating: 4.0
* Avg Completion Rate: 64.12%
* Completion Rate: 38.31%
* Dropout Rate: 16.89%

---

## 🔍 Key Insights

* Enrollment is consistently distributed across months with slight seasonal variation 
* Mobile devices dominate platform usage compared to desktop and tablet
* Certain categories like Business and Data Science drive higher enrollments
* Dropout rates vary across categories, indicating engagement gaps
* Beginner courses attract the highest enrollments compared to advanced levels
* Regional distribution shows strong participation from states like Bihar, West Bengal, and Delhi
